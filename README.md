# Verdoliva Eco Park

Verdoliva Eco Park is a responsive eco-adventure wildlife park website developed for the Web Technologies module at Solent University.

The project focuses on modern web development fundamentals using Node.js, Express, EJS templating, SQLite databases, and responsive front-end design principles.

## Features

* Responsive wildlife park website
* Dynamic habitat and experience pages
* SQLite database integration
* Contact form with validation
* AJAX-powered search and filtering
* Interactive wildlife-themed activity page
* Events system with filtering and dynamic content
* Accessibility and responsive design considerations

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* EJS
* SQLite3

## Project Goals

This project was developed to demonstrate:

* Full-stack web development fundamentals
* Client-side and server-side integration
* Database-driven content
* AJAX and dynamic page updates
* Responsive web design
* Accessibility and web standards compliance

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run the server:

```bash
node index.mjs
```

Open in browser:

```plaintext
http://localhost:5000
```

## Academic Context

This project was created as part of the QHO431 Web Technologies module at Solent University.
